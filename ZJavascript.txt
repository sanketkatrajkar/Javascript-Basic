5️⃣ Operator Precedence

Operator precedence means → which operator executes first.

Higher precedence executes first.
Common precedence order (high → low):
() Parentheses
Unary operators → ++ -- ! typeof
Multiplication / Division / Modulus
Addition / Subtraction
Comparison operators
Logical AND (&&)
Logical OR (||)

Assignment operators (= += -=)

Example:
let result = 5 + 3 * 2;  // 3*2 = 6 → 5+6 = 11

6️⃣ Unary Operator
Unary operators work on one single operand.

Examples:
let x = 5;
console.log(++x);   // Pre-increment (6)
console.log(x--);   // Post-decrement (prints 6, becomes 5)

Types:

Increment → ++x, x++
Decrement → --x, x--
Unary plus → +x
Unary minus → -x
typeof → check type

delete → delete object property

7️⃣ Promises : the promise object represents the eventual comletion (or failure ) of an asychronous operation ans its resulting value

function savetoDb(data, success, failure) {
    let internetSpeed = Math.floor(Math.random() * 10) + 1;
    console.log(`Saving "${data}" with speed ${internetSpeed}...`);
    if (internetSpeed > 4) {
        success();
    } else {
        failure();
    }
}

// Usage (callback hell example)
savetoDb("apna college", () => {
    console.log("success : your data was saved");
    savetoDb("hello world", () => {
        console.log("success2: data2 saved");

            console.log("success3: data3 saved");
        }, () => {
            console.log("failure3 : weak connection");
        });
    }, () => {
        console.log("failure2 : weak connection");
    });
}, () => {
    console.log("failure: weak connection, data not saved");

Using the promises 

function savetoDb(data) {
    return new Promise((resolve, reject) => {
        let internetSpeed = Math.floor(Math.random() * 10) + 1;
        console.log(`Saving "${data}" with speed ${internetSpeed}...`);
        if (internetSpeed > 4) {
            resolve();
        } else {
            reject();
        }
    });
}

Promises

request.then(()=> {

})

})



// Using Promise chaining (much cleaner)

savetoDb("apna college")
    .then(() => {
        console.log("success : your data was saved");
        return savetoDb("hello world");
    })
    .then(() => {
        console.log("success2: data2 saved");
        return savetoDb("shradha");
    })
    .then(() => {
        console.log("success3: data3 saved");
    })
    .catch(() => {
        console.log("failure: weak connection, data not saved");
    });
